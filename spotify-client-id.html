Use your own Spotify Client ID

If the shared Spotify app reaches its user limit, you can create your own free Spotify app and use its Client ID.

1. Create a Spotify app

  1. Open developer.spotify.com/dashboard ↗.
  2. Log in with your Spotify account.
  3. Click Create app.
  4. Use any app name, for example Študent naj bo.
  5. For the redirect URI, add the URL where you use the app.

For the hosted app, add:

https://rokroskar.github.io/student-naj-bo/

For local development, also add:

http://localhost:8000/

2. Copy the Client ID

  1. Open your Spotify app in the dashboard.
  2. Go to Settings.
  3. Copy the Client ID.

You do not need the Client Secret. Do not paste a Client Secret into this app.

3. Save it in this browser

Paste your Client ID below. It is stored only in this browser's localStorage.
